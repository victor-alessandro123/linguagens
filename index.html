Linguagens de Programação

Descubra o poder de cada linguagem

P5.js

O que é?

P5.js é uma biblioteca JavaScript que torna a programação criativa mais acessível, especialmente para artistas, designers e educadores.

Capacidades:

  • Criação de gráficos e animações interativas
  • Desenvolvimento de jogos simples
  • Visualização de dados
  • Arte generativa
  • Experiências interativas na web

JavaScript

O que é?

JavaScript é uma linguagem de programação versátil que roda no navegador e no servidor (Node.js).

Capacidades:

  • Desenvolvimento web front-end e back-end
  • Aplicações web interativas
  • APIs e serviços web
  • Aplicativos móveis (React Native)
  • Automação e scripts

Python

O que é?

Python é uma linguagem de programação de alto nível, conhecida por sua simplicidade e versatilidade.

Capacidades:

  • Inteligência Artificial e Machine Learning
  • Análise de dados e ciência de dados
  • Automação de tarefas
  • Desenvolvimento web (Django, Flask)
  • Scripts e ferramentas de produtividade

C#

O que é?

C# é uma linguagem moderna e orientada a objetos desenvolvida pela Microsoft.

Capacidades:

  • Desenvolvimento de aplicações Windows
  • Jogos (Unity)
  • Aplicações web (ASP.NET)
  • Aplicativos móveis (Xamarin)
  • Desenvolvimento de software empresarial

Java

O que é?

Java é uma linguagem de programação robusta e multiplataforma, conhecida por sua portabilidade.

Capacidades:

  • Aplicações empresariais
  • Desenvolvimento Android
  • Aplicações web (Spring)
  • Sistemas distribuídos
  • Desenvolvimento de jogos
